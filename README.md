## Task 1. Caesar cipher CLI tool

1. cd ./caesar-cipher-cli
2. npm instal
3. node my_caesar_cli -a encode -s 7 -i "./input.txt" -o "./output.txt"


CLI options:

1.  **-s, --shift**: a shift
2.  **-i, --input**: an input file
3.  **-o, --output**: an output file
4.  **-a, --action**: an action encode/decode
